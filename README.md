AI CAD Battleground Streamlit App
By Tyler Garn

GitHub Repository: https://github.com/twgs-ai/cad-battleground
Live Streamlit app URL: https://cad-battleground.streamlit.app/

	This app allows a user to write a prompt to generate a CAD file with 3 different lightweight Ais using the openscad library. This allows one to directly compare the performance of each ai and to download whichever .stl file they think is most accurate to their prompt. The ultimate goal of this app would be to add many more ai models to choose from with different model sizes, other cad libraries and a leaderboard showing which models with which libraries consistently generate more optimal results. The primary issues are the models struggling to write CAD code that is renderable. The small models also struggle with generating any shapes that are even fairly complicated.

	Prompts I suggest: ball, Rubik’s cube or a traffic cone.
